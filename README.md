# wekzs.com

Personal site. Built with [Astro](https://astro.build), deployed on Vercel.

---

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env` and add your `HEVY_API_KEY`. Without it the training chart uses mock data.

---

**Stack** — Astro 7, MDX, View Transitions, Dracula palette, Inter + JetBrains Mono

**Hosting** — Vercel (push to main → deploys). Monthly Hevy data refresh via GitHub Actions + Vercel Deploy Hook.
